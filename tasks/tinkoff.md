# Make a tinkoff website

### Reqiurements
1. Сайт является разводящей страницей на продукты компании
2. Сайт индексируется
3. Сайт сохраняет всю информацию, которую пользователь ранее вводил
4. Главная страницы состоит из информационных блоков, 
   посвяженных каждому продукту, во всю ширину экрана, один под другим, 
   а так же из ссылок на страницы, посвяженные каждому продукту в отдельности
5. При каждом следующем заходе на сайт, в самом верху пользователю 
   показывается инфоблок того продукта, в
   которому пользователь проявил наибольший интерес
